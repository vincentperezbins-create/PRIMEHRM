# PRIMEHR Manual Screenshots

Place screenshots in the folder for the role manual you want to generate:

- `manual_images/admin/`
- `manual_images/office_head/`
- `manual_images/school_head/`
- `manual_images/staff_user/`

Recommended image format: `.png`, `.jpg`, or `.jpeg`.

The generator first looks for a matching page name, then falls back to numbered order.

Examples:

- `login-page.png`
- `admin-dashboard.png`
- `user-list.png`
- `apply-leave.png`
- `01-login-page.png`
- `02-my-submission.png`
